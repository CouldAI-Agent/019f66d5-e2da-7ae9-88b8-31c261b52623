# Presentation Maker

A cross-platform Flutter application for creating and viewing presentations.

## Overview
Presentation Maker allows you to build slide decks directly on your device. You can add text, bullet points, and images to your slides, organize them, and present them in a dedicated viewer mode.

## Features
- **Dashboard**: View and manage your presentation decks.
- **Editor**: Create and edit slides with multiple content blocks.
- **Content Types**: Support for headings, paragraphs, bulleted lists, and image URLs.
- **Viewer**: Full-screen presentation mode to show off your slides.
- **Responsive**: Works across mobile, tablet, desktop, and web.

## Setup
Ensure you have Flutter installed.

```bash
flutter pub get
flutter run
```
